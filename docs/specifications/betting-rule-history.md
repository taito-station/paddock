# 買い方ルールの決定根拠・棄却記録・バックテスト履歴

現行の買い方ルールが「なぜこの形なのか」を残すアーカイブ。**運用ルール本体ではない**ので予想実行時に読む必要はなく、ルール変更を検討する時にだけ参照する。

- **現行ルール本体**: プロジェクト `CLAUDE.md` の「買い方ルール」セクション（毎セッション読込）。
- このドキュメントは、過去に検討して**棄却した改善案**と、ADR に未収録の**バックテスト/実績ログ**を保管する。棄却済みの案を再提案して再検証する無駄を防ぐのが目的。

## 棄却記録

### ① 混戦判定にオッズ条件を追加（棄却 / 2026-06-20 / ADR 0028）

「◎の単勝が X 倍以上なら混戦扱いにする」案。混戦判定にオッズ条件を足すと回収が改善するかを検証した。

- 検証: 71R（2026-05-30, 05-31, 06-13, 06-14）を `scripts/predict-check/konsen_backtest.py` で評価。
- 結果: **box-off（混戦なし）84.0% ≧ baseline（band≥4）83.9%**。オッズ条件は baseline を上回る閾値が一つも無かった。
- 追加発動した閾値（odds≥3.0/3.5 × band≥2/3）はいずれも回収を悪化させ **79.5〜81.5%**。唯一 odds≥4.0 は葵S（◎3.8倍）が閾値未満で追加発動 0 件 → **baseline 83.9% に退化（改善なし）**。band≥2 まで緩めると 13〜20R を混戦化し −3〜4pt。動機だった葵S は band=2 でボックスが組成不能（3頭未満）となり ¥1,500 が空振り。
- **結論**: オッズ条件は入れない。現行の `band≥4`（model 勝率 0.70 倍以上が ◎含め 4 頭以上）を維持。エンハンスは「回収を改善する証拠」が出てから入れる。
- 詳細: [docs/adr/0028-konsen-odds-trigger-rejected.md](../adr/0028-konsen-odds-trigger-rejected.md)（関連 Issue: #180）

### ② 混戦時の 3連複の相手拡大（top5 → top7/全頭）（棄却 / 2026-06-20 / ADR 0030）

混戦時に 3連複の相手を top5 より広げる案。函館10R（2026-06-20）で ◎④1着・▲⑥2着・3着⑤（model 7番手/25倍）を取り逃したのが動機。

- 検証: 105R（うち混戦 90R）。
- 結果: 的中率は 14%→22〜26% に向上するが、**回収率は全設定で低下**。
  - 混戦 ROI: 61.0% → 54.8〜57.5%
  - 全体 ROI: 87.7% → 82.4〜84.7%
- 均等重みでも改善せず。広げるほど外れ目に資金が分散し、当たり 1 点が薄くなる（典型的な「高的中・低配当」）。
- **結論**: 現行 top5 を維持。単発の取りこぼしで恒久ルール化しない。
- 詳細: [docs/adr/0030-konsen-trio-partner-width-rejected.md](../adr/0030-konsen-trio-partner-width-rejected.md)

## バックテスト/実績ログ（ADR 未収録）

ADR 化していないが、現行ルール（確率配分・混戦ボックス自動追加）の効果を裏付ける実測記録。数値はセッション時の実測ログに基づく当時の集計値で、再現スクリプトは未整備。

### 検証サマリ: 2026-05-31 主要 6R

配分方式を段階的に変えた時の回収率の比較。

| 配分方式 | 回収率 |
|---|---|
| 手組み ¥2,000/R | 81.7% |
| ¥5,000/R 確率配分 | 94.8% |
| ＋混戦ボックス自動追加 | 103.1%（+¥940） |

- 表の「+¥940」は最終構成（混戦ボックス追加）の純益（6R 合計 ¥30,000 に対し 103.1% 回収 ≒ +¥930、端数込み）。各方式の差分ではなく最終構成の絶対損益を指す。
- 白百合S: ◎⑧が着外だったが、混戦ボックスが①④⑦（2,500円）を捕捉し −5,000 → −2,500 に改善。
- 示唆: 確率配分と混戦ボックスはいずれも回収率を押し上げた。

### 実績ログ: 2026-06-20

- 21R 中 +EV は函館12R（◎④1.7倍 / model 35% / ROI〜125%）の **1 レースのみ**。
- 当該レースを ¥5,000 → ¥10,000 に増額し、**回収 222% / +¥12,160**。
- 断然人気（東京10R 1.7倍 80% 等）は全部 −EV で見送り、結果として正解だった。
- 示唆: +EV を厳選して増額する現行方針（レース選択基準）が機能した一例。
