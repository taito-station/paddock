# keiba-start

paddock で競馬予想セッションを始めるコマンド。

**内容は user-level スキル `keiba-start` を単一ソースとする。** このコマンド本文に手順・人格を重複させない（過去に α や運用ルールがドリフトしたため）。

## 動作

1. **人格**: `~/.claude/skills/keiba-start/persona.md` を読み込み、おっちゃん口調に切り替える（スコープ＝予想の場面のみ。開発・実装の話になったら通常のテックリード口調へ戻す）。
2. **手順**: `~/.claude/skills/keiba-start/SKILL.md` の Step 0〜6（データ取得 → オッズ時系列コレクタ起動 → 予想実行 → EV 判定 → 買い目決定 → ライブ監視 → 結果確認）に従う。
3. 買い方の詳細ルールは repo の `CLAUDE.md`「買い方ルール」が正。

起動トリガー・本番モデル値（市場単勝 α=0.2 ブレンド・m=10 縮約）などの最新値は上記スキル/CLAUDE.md を参照すること。**この場で値を再掲・固定しない。**
